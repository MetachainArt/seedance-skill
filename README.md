# Seedance 2.0 영상 프롬프트 스킬

Claude Code용 Seedance 2.0 AI 영상 프롬프트 생성 스킬입니다.

## 설치

```bash
claude skill install <이 리포 URL>
```

또는 `.claude/skills/seedance/SKILL.md` 파일을 직접 프로젝트에 복사하세요.

## 기능

- Seedance 2.0 플랫폼에 바로 사용 가능한 영상 프롬프트 생성
- 멀티모달 참조(@이미지, @영상, @오디오) 지원
- 10가지 생성 모드 지원 (순수 텍스트, 일관성 제어, 카메라워크, 특수효과, 스토리, 영상 연장, 사운드, 원테이크, 영상 편집, 음악 싱크)
- 초장 영상 분할 연결 전략
- 카메라 언어 / 스타일 / 감정 어휘 사전 내장
- 흔한 실패 패턴 및 트러블슈팅 가이드

## 사용법

Claude Code에서 다음과 같이 말하면 자동으로 스킬이 활성화됩니다:

- "영상 프롬프트 만들어줘"
- "Seedance 프롬프트 생성"
- "무협 전투 장면 영상 만들고 싶어"
- "광고 영상 프롬프트"
